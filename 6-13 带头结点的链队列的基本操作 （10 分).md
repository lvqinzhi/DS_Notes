# 6-13 带头结点的链队列的基本操作 （10 分)
实现链队列的入队列及出队列操作。


## 函数接口定义：
```c
Status QueueInsert(LinkQueue *Q,ElemType e)；

Status QueueDelete(LinkQueue *Q,ElemType *e)；
```


其中 Q 和 e 都是用户传入的参数。 


### LinkQueue 的类型定义如下：
```c
typedef int ElemType; 
typedef struct LNode
{
    ElemType data;
	struct LNode * next;
}LNode,*LinkList;
typedef struct
 {
   LinkList front,rear; /* 队头、队尾指针 */
 }LinkQueue;
 ```

{
	LinkList p;
	if(!L) return ERROR;

{
	int n=0;
	LinkList p;

	while(p)
	{
		n++;
		p=p->next;
	}
	return n;

## 代码：
```c
Status QueueInsert(LinkQueue* Q, ElemType e) {
	LinkList p;
	p = (LinkList)malloc(sizeof(struct LNode));
	p->next = NULL;
	p->data = e;
	Q->rear->next = p;
	Q->rear = p;
	return OK;
}

Status QueueDelete(LinkQueue* Q, ElemType* e) {
	LinkList p;
	p = (LinkList)malloc(sizeof(struct LNode));
	p = Q->front->next;
	Q->front->next = p->next;
	*e = p->data;
	free(p);
	return OK;
}
```
